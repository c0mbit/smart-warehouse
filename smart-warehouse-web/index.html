Smart Warehouse Management System

This project demonstrates the use of a microservice architecture (Java Spring Boot) and machine learning (Python FastAPI) to optimize warehouse operations and item slotting.

Get Started

How it works?

1. Create a new order in the "Order Assembly" tab.
2. The system saves the transaction to the PostgreSQL database.
3. The Python microservice analyzes the order history to find hidden patterns.

Artificial Intelligence

In the "Topology & AI" tab, you will see the warehouse grid. The algorithm provides slotting recommendations (Market Basket Analysis) to place frequently co-purchased items closer together, reducing picker travel time.
